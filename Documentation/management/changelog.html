    • Time Tracker was updated here

Version 0.3.2 - 2021-04-13
  • Changed a form item for 4.4.3.4 - Update Successful, where the redirect link, will be a button labeled, “Redirect”
  • Changed the Navigation sidebar where you can toggle its' display upon clicking the sidebar button (on top left corner)
  • Changed where the header bar is slightly larger (vertical height), to fit the sidebar button
  • Pages added

  • Document changes
    • SPMP Document has been updated to Version 0.6.1

Version 0.3.1 - 2021-04-04
  • Removed the attribute team_name from the Team Object, as our team decided to not support adding of Team Name. (Section Entity Object - 2. Team)
  • Document changes
    • Design Document has been updated to Version 0.2
    • Analysis Document has been updated to Version 0.4
    • Requirements Document has been updated to Version 0.4
    • SPMP Document has been updated to Version 0.6
    • Specifications Document has been updated to Version 0.4

Version 0.3 - 2021-03-30
  • Encyclopedia changed from 151 Pokémon to 150 Pokémon
    • Pokémon #151, "Mew", isn't normally available in the generation I Pokémon games. Mew will not be in the encyclopedia, reducing the amount of Pokémon to 150.

Version 0.2 - 2021-03-25
  • The site will now use the Bootstrap v4.5.3 Framework
    • All pages are using Bootstrap
    • Purpose of the change is for ease of reading and sake of modernization
    • The framework can help assist with the production of the Encyclopedia

Version 0.1 - 2021-03-20
  • Changed how dates are displayed from MM/DD/YYYY to YYYY-MM-DD, the ISO 8601 date standard
    • Ease of understanding of dates as 03/05/1970 could be March, 5th 1970 or May 3rd 1970.

  • Project name changed from Pokédex Encyclopedia to Pokémon Encyclopedia
    • The subject of the encyclopedia is not a Pokédex but Pokémon.
